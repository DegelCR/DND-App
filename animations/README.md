# Fumble animation asset

Download **Devil D20 Fumble** by Holli Lozinguez from LottieFiles:

https://lottiefiles.com/free-animation/devil-d20-fumble-rEH73ips20

## Steps

1. Open the link above in your browser
2. Click **Download as** → **Optimized Lottie JSON** (~32 KB)
3. Save the file here as:

```
public/animations/devil-d20-fumble.json
```

4. Restart the dev server (`npm run dev`) if it was running
